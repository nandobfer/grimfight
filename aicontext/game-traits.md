# Game Traits

## Features

### Arcanist
Arcanist é uma trait jogável registrada como `Arcanist`, inicialmente composta apenas por `freud`. A trait aparece para personagens da composição, mas só fica ativa quando o time possui personagens únicos suficientes para alcançar seus estágios.

Com 2 Arcanists ativos, cada unidade Arcanist ganha `5%` do próprio AP base a cada 5 segundos durante o combate. Com 4 Arcanists ativos, o mesmo ganho ocorre a cada 2 segundos. O ganho usa timer por unidade e só acumula enquanto a cena está no estado `fighting`.
